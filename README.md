# First contat with remote repository

First step I take was install GitHub Desktop and after that created a example repository with name:"Learning-Git-and-GitHub" and add a README.md to write about the repository and what they will bring in.

After that I click on green button "code" and clone the https method for this repository and after that opening the GitHub Desktop  I add the repository with method clone and with URL, this will create a clone repo, inside I can code whenever I want, but to make a change in main repository I will need the permission of owner of repository to accept my changes, if the changes was accepted will appear in the remote repository where is located in GitHub. It can be public or private where only the owner can see the repo.

After write what I need the next step is to commit for this I create the modification I make in repository and make a commit to main remote repository.
